# ToolSmith

Post-trained small-LLM tool-calling agent. Qwen3-4B-Instruct-2507 + LoRA SFT + step-level GRPO
(verifiable rewards), evaluated against a deterministic 12-tool travel-ops sandbox.

See `TOOLSMITH_BUILD_INSTRUCTIONS.md` for the task source and `ToolSmith_Project_Plan.md` for
architecture rationale.
